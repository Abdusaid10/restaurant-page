## Run Locally
To get a local copy up and running follow these simple example steps.

> Clone project to your local machine
```bash
git clone https://github.com/Abdusaid10/restaurant-page/
cd restaurant-page
```
> Install dependencies
```bash
npm install
```
> Build project
```bash
npm build
```
> Run Locally
```bash
npm start
```
